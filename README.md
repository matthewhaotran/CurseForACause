# CurseForACause

A web application that mimics a curse jar.  If you catch yourself cursing, add money to the jar.  At a certain point, donate it to a good cause!

## Live demo of the application

https://curseforacause.herokuapp.com

## How To Use

Clone repository

```
npm install
reload
```
